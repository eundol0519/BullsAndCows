# ⚾ 숫자 야구 게임(Bulls And Cows) ⚾

## 👉 게임 룰
- 게임을 진행할 숫자 갯수를 선택한다. (radio 버튼)
- 숫자를 랜덤으로 생성 받는다. (중복 X)
- 게임을 시작한다.
- 사용자가 숫자를 입력하면 정답 숫자와 사용자가 입력한 숫자를 비교하여
    - 숫자 포함 / 위치 일치 > 1 Strike
    - 숫자 포함 / 위치 불일치 > 1 Ball
    - 숫자 미포함 > 1 Out
- 10 라운드까지 진행하고, 그때까지 정답을 맞추지 못하면 게임 실패
- 10 라운드 이내에 정답을 맞추면 게임 성공
